# Video Game Sales Analysis

## Overview
Analyze video game sales data to uncover trends in consumer preferences, market demand, and genre popularity. Insights help developers, publishers, and stakeholders make data-driven decisions.

## Dataset
**Source:** [vgsales.csv on Kaggle](https://www.kaggle.com/datasets/gregorut/videogamesales)  
Includes game attributes, platform, genre, publisher, regional sales, and global sales.

## Methodology
- Exploratory Data Analysis (EDA)  
- Regression, Decision Tree, and Random Forest models to predict global sales  
- Visualizations using Matplotlib, Seaborn, and Power BI dashboards  
- Data preprocessing with Pandas and NumPy  

## Key Insights
- Genre popularity varies by region  
- Platform preferences influence sales trends (PS, Xbox, Wii, etc.)  
- Regional sales patterns inform marketing and localization strategies  
- Historical trends guide future market projections
